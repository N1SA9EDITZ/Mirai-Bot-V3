# 📌 Introduction
KuRuMi Bot is an upgraded Messenger chatbot project based on Mirai Bot system.  
This project is modified and improved for better performance, stability, and customization.

---

# ⚙️ Requirements

- Node.js **v20+**
- Git installed
- 1–2 GB free storage
- Basic knowledge of JavaScript / Node.js
- Facebook account (recommended: not your main account)
- Platform tools:
  - Windows: Build Tools
  - Linux: Python3
  - Android: Termux
